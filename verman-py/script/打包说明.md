# VerMan 打包说明

## 快速开始

### 1. 打包程序
```bash
python script/build_exe_simple.py
```

### 2. 安装右键菜单
```bash
script/install_exe_menu.bat
```

### 3. 卸载右键菜单
```bash
script/uninstall_exe_menu.bat
```

## 详细说明

请参考 `script/README.md` 获取完整的打包和部署说明。

## 文件位置

- 打包后的exe文件：`dist/VersionManager.exe`
- 所有打包脚本：`script/` 目录
- 使用文档：`script/README.md`
