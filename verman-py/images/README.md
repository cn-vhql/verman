# 收款码图片说明

这个目录用于存放VerMan VIP版本的收款码图片。

## 文件说明

### alipay_qr.png
- 支付宝收款码图片
- 建议尺寸：200x200 像素
- 格式：PNG
- 请将您的支付宝收款码保存为此文件名

### wechat_qr.png
- 微信收款码图片
- 建议尺寸：200x200 像素
- 格式：PNG
- 请将您的微信收款码保存为此文件名

## 如何添加收款码

1. 打开支付宝/微信，找到您的收款码
2. 截图保存收款码
3. 使用图片编辑工具调整大小为200x200像素
4. 分别保存为 `alipay_qr.png` 和 `wechat_qr.png`
5. 放在此目录中

## 注意事项

- 确保收款码清晰可识别
- 建议使用真实的收款码
- 图片大小建议不要超过50KB
- 支持PNG、JPG格式
