## 🚀 使用指南

### 1. 准备收款码

1. **支付宝收款码**
   - 打开支付宝App → 收钱
   - 截图保存收款码
   - 调整为200x200像素
   - 保存为 `images/alipay_qr.png`

2. **微信收款码**
   - 打开微信App → 收付款 → 二维码收款
   - 截图保存收款码
   - 调整为200x200像素
   - 保存为 `images/wechat_qr.png`

### 2. 运行激活码生成器

```bash
python activation_generator.py
```

功能包括：
- 生成单个激活码
- 批量生成激活码
- 查看统计信息
- 搜索和导出记录

### 3. 测试系统

```bash
python simple_vip_test.py
```

验证所有功能是否正常工作。

### 4. 启动软件

```bash
python main.py
```

软件将自动显示VIP功能限制，引导用户升级。

## 🔧 配置说明

### 密钥配置
默认密钥：`verman_vip_secret_key_2024`
位置：`license_manager.py` 和 `activation_generator.py`

如需更换密钥：
1. 修改两个文件中的 `secret_key`
2. 重新部署软件
3. 所有现有激活码将失效
